# El Excel

A simple `excel-like` personal finance app.


# Features

- [ ] Dark mode
- [ ] Installable `PWA`
- [ ] Offline mode
- [ ] Data sync with `Google Drive`
- [ ] Easy `transfers` between accounts
- [ ] Local & sync workspaces
- [ ] Export and import [`.json`, `.csv`, `.xlsx`, `.pdf`]
